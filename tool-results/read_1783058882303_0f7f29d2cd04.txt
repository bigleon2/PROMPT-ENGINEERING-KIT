   157→
   158→### Types supportes
   159→PDF, XPS, EPUB, CBZ, images (PNG, JPEG, TIFF, BMP, GIF, etc.), texte brut.
   160→
   161→### Methodes d'ouverture
   162→
   163→```python
   164→import fitz  # PyMuPDF
   165→
   166→# Ouverture standard
   167→doc = fitz.open("fichier.pdf")
   168→
   169→# Ouverture depuis URL distante
   170→doc = fitz.open("https://example.com/document.pdf")
   171→
   172→# Ouverture en tant que texte
   173→doc = fitz.open("texte.txt")
   174→
   175→# Reconnaisseur de fichiers (extension incorrecte)
   176→# PyMuPDF detecte le vrai type meme si l'extension est fausse
   177→
   178→# Ouverture depuis flux (bytes)
   179→doc = fitz.open(stream=data, filetype="pdf")
   180→```
   181→
   182→### Options de open()
   183→- `stream` : ouvrir depuis des bytes en memoire.
   184→- `filetype` : forcer le type ("pdf", "xps", "epub", "cbz", "img").
   185→- `rect` : definir la taille de la page.
   186→- `fontsize` : taille de police pour le texte.
   187→- `lang` : langue pour le texte (OCR).

   274→## Domaine 4 — Texte — Extraction, Analyse, Insertion (PyMuPDF / PyMuPDF4LLM)
   275→
   276→### 4.1 Extraction de texte
   277→
   278→```python
   279→import fitz
   280→
   281→doc = fitz.open("fichier.pdf")
   282→
   283→# Extraction de tout le texte du document
   284→text = doc[0].get_text()  # texte brut
   285→
   286→# Extraction en mode "marquage" (HTML-like)
   287→text_html = doc[0].get_text("html")
   288→
   289→# Extraction en JSON
   290→text_json = doc[0].get_text("json")
   291→
   292→# Extraction en RAWJSON (plus detaille)
   293→text_raw = doc[0].get_text("rawjson")
   294→
   295→# Extraction en XML
   296→text_xml = doc[0].get_text("xml")
   297→
   298→# Extraction en XHTML
   299→text_xhtml = doc[0].get_text("xhtml")
   300→
   301→# Extraction par rectangle (zone specifique)
   302→rect = fitz.Rect(100, 100, 500, 300)
   303→text_rect = doc[0].get_text("text", clip=rect)
   304→
   305→# Extraction dans l'ordre naturel de lecture
   306→blocks = doc[0].get_text("blocks")  # liste de blocks (x0, y0, x1, y1, text, block_no, block_type)
   307→words = doc[0].get_text("words")    # liste de mots (x0, y0, x1, y1, word, block_no, line_no, word_no)
   308→dicts = doc[0].get_text("dicts")    # structure hierarchique
   309→```
   310→
   311→### 4.2 Extraction de tables
   312→```python
   313→# Extraction de tables
   314→tables = doc[0].find_tables()
   315→for table in tables.tables:
   316→    # table.extract() → liste de lignes, chaque ligne est une liste de cellules
   317→    data = table.extract()
   318→    # table.header → ligne d'en-tete detectee
   319→    # table.col_count, table.row_count → dimensions
   320→```
   321→
   322→### 4.3 Extraction de paires cle-valeur
   323→```python
   324→# Via PyMuPDF4LLM
   325→import pymupdf4llm
   326→
   327→kv = pymupdf4llm.get_key_values("fichier.pdf")
   328→# Retourne une liste de paires cle-valeur detectees automatiquement
   329→```
   330→
   331→### 4.4 Analyse des polices et couleurs
   332→```python
   333→# Analyse des fontes utilisees
   334→fonts = doc[0].get_fonts()
   335→# Retourne la liste des fontes avec leurs proprietes
   336→
   337→# Extraction du texte en couleur
   338→text_color = doc[0].get_text("colors")
   339→# Retourne le texte avec les informations de couleur
   340→
   341→# Flags d'extraction avances
   342→text = doc[0].get_text("text", flags=fitz.TEXT_PRESERVE_WHITESPACE | fitz.TEXT_PRESERVE_LIGATURES)
   343→```
   344→
   345→### 4.5 Insertion et ecriture de texte
   346→```python
   347→# Inserer du texte
   348→page = doc[0]
   349→page.insert_text((100, 100), "Hello World", fontsize=12)
   350→
   351→# Ecrire des lignes
   352→writer = fitz.TextWriter(page.rect)
   353→writer.fill_textbox(fitz.Rect(50, 50, 500, 300), "Texte long a insérer dans une boite...")

   361→PyMuPDF gere les ligatures (fi, fl, ff, ffi, ffl, etc.) via les flags d'extraction. Utiliser `fitz.TEXT_PRESERVE_LIGATURES` pour les preserver ou les convertir.

   365→## Domaine 5 — Images (PyMuPDF / Pro)
   366→
   367→### 5.1 Creation d'images a partir de pages
   368→```python
   369→# Rendu d'une page en image (pixmap)
   370→page = doc[0]
   371→pix = page.get_pixmap(dpi=150)  # resolution en DPI
   372→pix.save("page.png")
   373→
   374→# Augmentation de resolution
   375→pix = page.get_pixmap(matrix=fitz.Matrix(2, 2))  # zoom x2
   376→```
   377→
   378→### 5.2 Pixmaps partiels (clips)
   379→```python
   380→# Extraire une zone specifique de la page
   381→clip = fitz.Rect(100, 100, 400, 300)
   382→pix = page.get_pixmap(clip=clip)
   383→```
   384→
   385→### 5.3 Extraction d'images embarquees
   386→```python
   387→# Lister toutes les images du document
   388→for page_num in range(len(doc)):
   389→    page = doc[page_num]
   390→    images = page.get_images(full=True)
   391→    for img_index, img in enumerate(images):
   392→        xref = img[0]
   393→        base_image = doc.extract_image(xref)
   394→        # base_image["image"] → bytes de l'image
   395→        # base_image["ext"] → extension (png, jpeg, etc.)
   396→        # base_image["width"], base_image["height"] → dimensions
   397→        # base_image["colorspace"] → espace de couleur
   398→```
   399→
   400→### 5.4 Gestion des masques d'image
   401→```python
   402→# Extraire une image avec son masque (SMask)
   403→xref = img[0]
   404→pix = fitz.Pixmap(doc, xref)
   405→if pix.alpha:  # l'image a un canal alpha / masque
   406→    pix = fitz.Pixmap(pix, 0)  # supprimer le canal alpha
   407→```
   408→
   409→### 5.5 Creation d'un PDF contenant toutes les images
   410→```python
   411→# Extraire toutes les images et les mettre dans un nouveau PDF
   412→new_doc = fitz.open()
   413→for page_num in range(len(doc)):
   414→    page = doc[page_num]
   415→    for img in page.get_images(full=True):
   416→        xref = img[0]
   417→        base_image = doc.extract_image(xref)
   418→        img_doc = fitz.open(stream=base_image["image"], filetype=base_image["ext"])
   419→        new_doc.insert_pdf(img_doc)
   420→new_doc.save("all_images.pdf")
   421→```
   422→
   423→### 5.6 Interface avec NumPy
   424→```python
   425→import numpy as np
   426→
   427→pix = page.get_pixmap()
   428→# Convertir pixmap en array NumPy
   429→samples = pix.samples  # bytes bruts
   430→np_array = np.frombuffer(samples, dtype=np.uint8).reshape(pix.height, pix.width, pix.n)
   431→
   432→# Inverse : creer un pixmap depuis un array NumPy
   433→pix = fitz.Pixmap(np_array)
   434→```
   435→
   436→### 5.7 Ajout d'images dans un PDF
   437→```python

   439→page = doc[0]
   440→rect = fitz.Rect(100, 100, 400, 300)
   441→page.insert_image(rect, filename="image.png")
   442→
   443→# Inserer depuis des bytes

   524→## Domaine 8 — Stories — Moteur de Composition Avance (PyMuPDF)
   525→
   526→Le systeme Stories est le moteur de mise en page avance de PyMuPDF. Il permet de creer des PDF complexes avec :
   527→
   528→### Fonctionnalites principales
   529→- Ajout de texte formate (HTML/CSS).
   530→- Integration d'images.
   531→- Lecture HTML/CSS externe.
   532→- Templates pour bases de donnees.
   533→- Integration avec PDF existants.
   534→- Layouts multi-colonnes.
   535→- Zones "no-go" (exclusion).
   536→- Sortie de tableaux HTML.
   537→- Grilles.
   538→- Generation de table des matieres.
   539→- Layout stabilise (`write_stabilized`).
   540→- Creation de liens PDF (`write_stabilized_links`).
   541→- Affichage de listes JSON.
   542→
   543→### Exemple d'utilisation
   544→```python
   545→import fitz
   546→
   547→story = fitz.Story(html="<h1>Titre</h1><p>Contenu avec <b>gras</b>.</p>")
   548→
   549→# Integration dans un PDF existant
   550→doc = fitz.open()
   551→page = doc.new_page()
   552→rect = page.rect
   553→
   554→# Ecriture avec layout stabilise
   555→writer = fitz.TextWriter(page.rect)
   556→story.write(writer, rect)
   557→writer.write_text(page)
   558→
   559→# Avec zones "no-go"
   560→mediabox = page.mediabox
   561→where = mediabox  # zone d'ecriture
   562→story.write(writer, where, exclude=fitz.Rect(0, 0, 50, 50))  # exclure un coin
   563→
   564→# write_stabilized : ecriture avec liens et TOC automatiques
   565→story.write_stabilized(doc, where, create_links=True)
   566→```
   567→
   568→### Generation de TOC
   569→```python
   570→# La methode write_stabilized peut generer automatiquement une table des matieres
   571→story.write_stabilized(doc, where, page_number=True)
   572→toc = doc.get_toc()  # recuperer la TOC generee
   573→```

   718→### 12.3 Texte manquant ou illisible
   719→- **Cause 1** : Le texte est une image (scan) → utiliser OCR (Pro).

   875→## Auto-Diagnostic
   876→
   877→A la fin de chaque tache, produire une **section de diagnostic** :
   878→
   879→### E.1 Qualite du document
   880→- **Lisible** : texte selectionnable, structure coherente.
   881→- **Partiellement lisible** : certaines pages scannees, tableaux mal structures.
   882→- **Corrompu** : erreurs xref, streams endommages.
   883→- **Scanne** : pas de texte selectionnable, OCR necessaire.
   884→
   885→### E.2 Qualite de l'extraction
   886→- **Complete** : tout le texte extrait, tables intactes.
   887→- **Partielle** : sections manquantes, images non extraites.
   888→- **Bruit** : artefacts, ligatures mal gerees, espaces incorrects.
   889→- **Ambigue** : colonnes fusionnees, en-tetes non detectes.
   890→
   891→### E.3 Problemes detectes
   892→- Calques invisibles (OCG desactives).
   893→- Texte non selectionnable (necessite OCR).
   894→- Tables mal structurees (cellules fusionnees).
   895→- Images manquantes (masque, embedding).
   896→- Polices non standardes (glyphes manquants).
   897→- Contenu dans des Form XObjects non traites.
   898→
   899→### E.4 Risques pour le RAG / LLM
   900→- Sections manquantes → perte de contexte.
   901→- Ambiguites dans les tables → donnees incorrectes.
   902→- Colonnes non detectees → texte melange.
   903→- Metadonnees absentes → pas de tri/filtrage.
   904→
   905→**Tu dois decrire honnetement les limites de ton extraction.**

  1015→| Extraire des images | PyMuPDF | `page.get_images()` + `extract_image()` | Avec metadonnees |
  1016→| PDF → SVG vectoriel | PyMuPDF | `page.get_svg_image()` | |
  1017→| HTML → PDF | PyMuPDF | `page.insert_html()` | Avec CSS |
